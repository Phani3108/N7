We engage the following sub-processors to run our products. Each is contractually bound to process personal data only as needed to deliver the service. We notify paying customers at least 30 days before adding a new entry below.

| Sub-processor | Purpose | Region |
|---|---|---|
| **Cloudflare** | CDN, DNS, DDoS protection | Global edge |
| **Amazon Web Services** | Primary hosting, object storage, KMS | US · EU · IN (per customer region) |
| **Stripe** | Payment processing for paid plans | US / EU (per billing address) |
| **Anthropic** | LLM inference for agents using Claude models | US |
| **OpenAI** | LLM inference for agents using GPT models | US |
| **Google Cloud** | LLM inference for agents using Gemini models | US / EU |
| **Deepgram** | Speech-to-text for voice-enabled products | US |
| **ElevenLabs** | Text-to-speech for voice-enabled products | US |
| **Plaid** | Bank data connectivity for TransactionWonder | US / EU / CA |
| **Twilio** | Voice and SMS infrastructure for Vexa | Global |
| **Sentry** | Error and performance monitoring | US / EU (per customer region) |
| **Resend** | Transactional email | US |

## How to object

If you object to our use of any sub-processor on the list above, email [privacy@number7.io](mailto:privacy@number7.io). If we cannot provide a mutually acceptable alternative, either party may terminate affected services.
